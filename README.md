# Sistema de Gestión de Inventario - CRUD MongoDB

Este es un proyecto educativo desarrollado con **Node.js**, **Express** y **MongoDB Compass** para la plataforma Unetieduca.

## Características
- Conexión a base de datos NoSQL (MongoDB).
- 5 Colecciones configuradas.
- CRUD completo (Crear, Leer, Actualizar, Eliminar).
- Consulta de stock crítico.

## Requisitos
- Node.js instalado.
- MongoDB Compass ejecutándose localmente.
- Thunder client para ejecucion

## Instrucciones de Ejecución
1. Clonar el repositorio:
   git clone (https://github.com/Junior3801/CRUD_MONGOBD_DS.git)
2. Instalar dependencias:
   npm install
3. Iniciar el servidor:
   npm start

El servidor estará disponible en: http://localhost:3000
